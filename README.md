# React Movie App

A simple movie search app built with React, Vite, and Appwrite.

## Features
- Search for movies
- View movie cards with poster and details
- See trending searches

## Tech Stack
- React
- Vite
- Appwrite

## Run locally
1. Install dependencies
   ```bash
   npm install
   ```
2. Start the app
   ```bash
   npm run dev
   ```

## Notes
This project uses Appwrite and requires your own environment variables.
